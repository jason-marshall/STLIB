STLib

Packages

Welcome to my C++ template library. This is a heterogeneous collection of projects primarily dealing with computational geometry, stochastic simulations, and static Hamilton-Jacobi equations. Below are the major packages. Follow a link to get documentation for a specific package.
Title Namespace Description
Algorithms and Data Structures ads Arrays (static, dynamic, and multi-dimensional), functors, iterators, priority queues, a half-edge data structure, tensors, a timer, and more.
Adaptive Mesh Refinement amr An Orthtree (quadtree, octree, etc.) data structure for adaptive mesh refinement on logically Cartesian meshes.
Containers container Classes for 1-dimensional and multi-dimensional arrays.
Closest Point Transform cpt Closest point and (signed and unsigned) distance transform to polygons in 2-D and triangle meshes in 3-D.
Computational Geometry geom Primitives, polytopes, trees, grids, orthogonal range queries, spatial indexing, and simplicial mesh optimization.
Concurrent algorithms concurrent Partitioning, task queueing, and communication algorithms.
Eulerian/Lagrangian Coupling elc The point-to-point communication in the Eulerian/Lagrangian coupling of a solid and fluid solver with the ghost fluid method.
Standard Library Extensions ext Extensions to the C++ standard library.
Level Set levelSet Level Set methods for implicit geometry.
LOR for Geometric DS lorg Locality of reference for geometric data structures.
Space-Filling Curves sfc Space-filling curves and orthant tries.
Molecular Surface Triangulations mst Triangulations of the surface of a dynamically changing molecule.
MPI Wrapper and Utilities mpi Functions and classes that ease MPI programming.
Numerical Algorithms numerical Random numbers, special functions, optimization, derivatives, interpolation.
Particle particle Computational geometry tools for particle methods: data layout, neighbor queries, partitioning, communication patterns.
Performance performance Timers and loggers for measuring the performance of serial and concurrent programs.
Single-Source Shortest-Paths shortest_path I compare the standard greedy algorithms with my greedier algorithms.
SIMD Programming simd Classes, functions, and constants that ease the task of SIMD programming.
Static Hamilton-Jacobi Equations hj I have implementations of Sethian's Fast Marching Method and the Marching with a Correctness Criterion algorithm in 2-D and 3-D.
Stochastic Simulations stochastic Exact and approximate stochastic simulation algorithms for a set of reactions.
