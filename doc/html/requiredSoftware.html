Required Software

Back

In order to use STLib, you will need some third-party software. Boost is required for most packages. Note that you don't need the compiled Boost libraries, just the headers. Catch is required for some unit tests. This is a pure header library, so no compilation is required. Both the geom/shape package and the implicit tau-leaping in stochastic use Eigen.

In order to compile sources that use STLib, these libraries must be in your include path. Here we give instructions for putting them in include in your home directory.

For Catch, get the latest single header version. Right click on Latest Master build and save the linked file to your Downloads folder. Then move the file to include. 

$ cd ~/include
$ cp ~/Downloads/catch.hpp .

Download the latest version of Boost and unpack it. Then copy the header files to include. 

$ cp -R ~/Downloads/boost_1_56_0/boost .

Download the latest version of Eigen and unpack it. Then copy the header files to include. 

$ cp -R ~/Downloads/eigen-eigen-1306d75b4a21/Eigen .

MS Windows

If you develop on MS Windows, then using STLib should be fairly easy. Just put it in a convenient location and put path/to/stlib in your include path when you compile. You will likely also need the Boost++ headers. The rest of this section is documentation for myself. I describe how to set up a development environment on Windows.

Install the Visual Studio Express Windows Desktop application. (This requires less disk space than the other applications.) Install the following: Windows PowerShell 4.0, Microsoft MPI, Python 2.7, SCons, Git, and SourceTree. Add C:\Program Files\Microsoft SDKs\MPI\Lib\x86 (or the x64 directory for 64-bit architectures) and C:\Python27\Scripts to the system Path environment variable.

Install Boost++. I recommend grabbing one of the installers with pre-compiled binaries instead of installing from source. Suppose that it is installed in C:\local\boost_1_58_0. Add the library directory to the system Path environment variable. For example, you might add the C:\local\boost_1_58_0\lib32-msvc-12.0 directory. In your home directory, create an include directory. There, add a junction to the Boost++ include directory. To do this, click the Start button and then right-click on Command Prompt and select Run as administrator. Navigate to the include directory and execute the following command. 

C:\Users\Sean\include>mklink /D boost C:\local\boost_1_58_0\boost

Download the single header file for Catch and put catch.hpp in the include directory. Download Eigen. Copy the Eigen directory, which contains the header files, to the include directory in your home directory.

After cloning stlib with SourceTree, add a junction to stlib in your include directory. 

C:\Users\Sean\include>mklink /D stlib C:\Users\Sean\dev\stlib\stlib

Incidentally, don't try using SharpKeys to swap the control, alt, and Windows keys. The program can override keys, but it can't swap them.
