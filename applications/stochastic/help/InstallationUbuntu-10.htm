Installation on Ubuntu 10

Select Applications→Ubuntu Software Center.

  • g++: The GNU C++ compiler.
  • python-wxversion: wxWidgets cross-platform C++ GUI toolkit.
  • python-matplotlib: Python based plotting system in style similar to Matlab.
  • python-scipy: Scientific Tools for Python.
  • python-sympy: Computer Algebra System (CAS) in Python.
Then follow the installation instructions to compile the solvers and launch the application.
