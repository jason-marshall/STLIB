dsmts-004-01

This model has a single species X and has two reactions with mass-action kinetics. The batch immigration reaction is 0 → 5 X with rate parameter Alpha = 1. The death reaction is X → 0 with rate parameter Mu = 0.2. Use the open button   in the tool bar to import the file examples/sbml/dsmts31/dsmts-004-01.xml. The model is imported correctly and the reactions are recognized as having mass-action kinetics.
