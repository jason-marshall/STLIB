Installation on Linux/Unix

For Linux or Unix, use the platform-independent distribution. You will need appropriate versions of Python, wxPython, matplotlib, and numpy, as well as a C++ compiler. There are platform-specific instructions for installing this software for RedHat 5.3, CentOS 5.2, and Fedora 10. I recommend using the current version of GNU GCC. Note that only Python versions 2.4.x through 2.6.x are currently supported. If you do not have the necessary Python packages installed, Cain will show an error message when you attempt to launch the application.

Download the platform-independent distribution and place it in a convenient location. Then uncompress the zip file.
unzip Cain.zip
Build the mass action solvers.
cd Cain
make
Then launch the GUI.
python Cain.py&
There are example files in the examples directory. You can open a file on startup by specifying the file as a command line argument. For example:
python Cain.py examples/cain/BirthDeath.xml&

To uninstall Cain, simply delete the Cain directory.
