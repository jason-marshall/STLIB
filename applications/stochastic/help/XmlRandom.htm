Random Numbers

The random element stores the states for the Mersenne twister random number generator. This state is a list of 625 integers: an array of 624 integers followed by an index into the array. The seed is used for generating new states. 

  <random seed="Integer">
    For each state:
      <stateMT19937>
        List of integers.
      </stateMT19937>
  </random>
